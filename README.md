Simulating speech delay. Also help from [rollup] and [pico]

[rollup]: https://rollupjs.org/guide/en/

## Run build locally

```bash
npm ci # only needs to be run once
npm run build # build:watch for fast turnarounds
```

## Serve files locally

```
npm run serve
```
